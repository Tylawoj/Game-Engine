Movement:

WSAD - XZ-axis movement
Left Shift - Move up
Left Ctrl - Move down

==============================================

The program was tested on a Live-USB version of Manjaro.
If it fails to run, build it using the CMake file.

If it fails to build on Linux, uncomment the LINK_DIRECTORIES line.
If it still fails to build... try with the SDL2, GL and OpenAL dependencies installed.
